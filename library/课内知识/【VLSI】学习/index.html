计算延迟

认识参数

意义计算
$t_{pdr}$rising propagation delayFrom input to rising output crossing VDD/2
$t_{pdf}$falling propagation delayFrom input to falling output crossing VDD/2
$t_{pd}$average propagation delay(tpdr+tpdf)/2
$t_r$rise timeFrom output crossing 0.2 VDD to 0.8 VDD
$t_f$fall timeFrom output crossing 0.8 VDD to 0.2 VDD

意义计算
$t_{cdr}$rising contamination delayFrom input to rising output crossing VDD/2
$t_{cdf}$falling contamination delayFrom input to falling output crossing VDD/2

contamination delay和propagation delay都是输入到输出的延迟，不同点在于不同输入情况，延迟不同。

门电路延迟

  • d: delay
  • f: effort delay (stage effort)
  • p: parasitic delay (Represents delay of gate driving no load)
  • g: logical effort (Measures relative ability of gate to deliver current)
  • h: electrical effort (Ratio of output to input capacitance)
